# Zoronal Assignment - Server

Backend API for the Zoronal assignment project built using Node.js, Express, and MongoDB.

## Features

- REST API architecture
- User authentication and authorization
- Database integration with MongoDB
- CRUD operations
- Error handling and validation
- Environment-based configuration
- CORS support

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- dotenv
